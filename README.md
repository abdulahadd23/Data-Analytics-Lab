# Data Analytics Lab — Spring 2026

A complete collection of hands-on lab work covering the core data analytics pipeline in Python: from setting up the environment to building a full visualization portfolio. Each lab folder contains a Jupyter Notebook with documented code, explanations, and outputs.

## Lab Breakdown

**Lab 1 — Environment Setup & First Steps**  
Anaconda installation, Jupyter Notebook configuration, library verification (pandas, NumPy, matplotlib, seaborn), basic Python operations, and GitHub repository initialization.

**Lab 2 — Python Data Structures & File I/O**  
Lists, dictionaries, tuples, and sets in action. Custom functions for data processing, reading/writing CSV, JSON, and text files, list comprehensions with performance benchmarks, and error handling patterns.

**Lab 3 — Numerical Computing with NumPy**  
Array creation (1D, 2D, 3D), indexing, slicing, reshaping, broadcasting, and element-wise operations. Statistical analysis on simulated stock price data, linear algebra (dot product, matrix multiplication, eigenvalues), random sampling, and a Monte Carlo Pi estimation.

**Lab 4 — Data Wrangling with Pandas**  
Loading data from CSV, Excel, and JSON. Full exploration using head(), tail(), info(), describe(). Boolean indexing, multi-column sorting, missing value handling (median imputation by class), and feature engineering (FamilySize, AgeGroup, FarePerPerson). Dataset: Titanic.

**Lab 5 — Visualization Portfolio**  
12 chart types built on simulated COVID-19 data: line plots, bar charts, histograms, scatter plots, box plots, violin plots, heatmaps, area charts, pie/donut charts, stacked bars, multi-panel dashboards, and pair plots. All customized with proper labels, legends, and color palettes.
